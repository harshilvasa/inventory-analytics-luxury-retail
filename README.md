# Multi-Store Inventory Analytics – Luxury Retail

This project analyzes inventory data across 4 stores and 40 SKUs in a luxury retail context using PostgreSQL and Tableau.

## Project Goals
- Calculate inventory accuracy by store
- Quantify shrinkage ($) by store, category, and SKU
- Evaluate RFID impact on inventory accuracy
- Identify top SKUs driving the majority of shrinkage (Pareto 80/20)

## Tools Used
- PostgreSQL | SQL
- Tableau
- Excel

## Folder Structure
- `data/` – CSV dataset
- `sql/` – SQL queries for each analysis
- `tableau/` – Dashboard screenshots

## Key Insights
- RFID SKUs show higher accuracy and lower shrinkage
- ~20% of SKUs account for ~80% of total shrinkage
- Leather Goods and Footwear categories drive most financial impact
